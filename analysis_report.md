# Sales Data Analysis Report

## Project Overview
This project analyzes a sales dataset using Python and Pandas to understand sales performance.

## Dataset Details
- Rows: 100
- Columns: 5

## Analysis Performed
- Total sales calculation
- Average sales calculation
- Best-selling product identification
- Handling missing values

## Key Findings
- Total Sales were calculated successfully
- Best-selling product was identified using groupby
- Missing values were handled using fillna()

## Conclusion
The analysis helped understand overall sales performance and identify top products.
